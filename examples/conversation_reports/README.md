# Human-AI Conversation Report Samples

This folder contains a synthetic sample pack for showing Spectrum's comprehensive conversation-report direction.

- `index.html` is a standalone gallery you can open in a browser.
- `sample_reports.json` contains the same sample report data in a structured form.

The four sample domains are:

- Customer support and issue resolution
- Appointment scheduling and reminders
- Outbound sales follow-up and qualification
- Billing, payment, and collections follow-up

The market scan in the HTML also calls out surveys, feedback, and post-service checks as the next common report mode to add.

These are demo calls, not real customer recordings. They are intentionally written to show how a report should drill into executive diagnosis, findings, likely causes, human experience, agent behavior, trust limits, and transcript evidence.
